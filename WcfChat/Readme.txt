Написать клиент-серверное приложение с использование WCF

Требования:

1. Сервис должен содержать как минимум 4 метода
2. Должно быть реализовано сохранение данных в БД (использовать Entity Framework, или Linq-to-Sql)
3. Класс (DataContract) не менее 3х свойств
4. Одна форма (WinForms или WPF)
5. Конфигуация должна храниться в config файлах
6. Обработка ошибок с помощью FaultException. Контракты ошибок(мин 2)
7. Минимум 2 конечные точки с разным Binding-ом
